A code agent that works inside your project

UtterIDE connects project context, code patches, SQL analysis, diagrams and review-first approvals in one VS Code workflow.

📝

Code Patches

Reads files, understands context, proposes diffs and waits for approval before applying changes.

🗄️

SQL & Databases

Explains schemas, analyzes slow queries, suggests indexes and masks sensitive data before cloud calls.

🐍

Python & Scripts

Creates helper scripts, refactors code and helps run checks without hiding what changed.

📊

⚖️

🧾

Run Ledger

See what was read, changed, approved, rejected, touched and exactly what it cost.

Local-First Architecture

No central UtterIDE backend. Your project memory, run logs and settings are stored locally. When you use a cloud AI provider, selected context is sent directly from your machine to that provider. Sensitive data masking is designed to reduce risk, but users should still review what is sent to cloud providers.
